## Requerimientos
- Node js
- npm

## Iniciar proyecto
- git clone git@github.com:RogerFranck/UnoBots.git
- cd UnoBots
- git checkout test (main es para servidor)
- npm i
- npm run build && npm run start
- En el navegador poner 'http://localhost:3000/'
- Ganar la partida!!
